DOCX Inline Formatting Demo

This document demonstrates inline run properties (w:rPr) applied directly in document.xml without using styles.xml. Each section shows the style name, a styled sample, the corresponding run XML, and a note about CSS equivalence or approximation.

Bold

Sample Text

Run XML:

<w:r>
 <w:rPr><w:b/></w:rPr>
 <w:t>Sample Text</w:t>
</w:r>

CSS mapping: Direct CSS: font-weight: bold;

Sample Text
