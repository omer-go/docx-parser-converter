DOCX Formatting Demo - Combinations, Paragraph Properties, Fonts

This document demonstrates combined inline run properties, paragraph properties affecting layout, and font/size variations, all applied directly in document.xml without styles.xml. Each section shows a header, a styled sample, the corresponding XML, and a CSS mapping or approximation.

Sample Text:

Sample Text

</w:r>
