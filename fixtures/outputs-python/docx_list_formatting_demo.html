DOCX List Formatting Demo (v5)

This document demonstrates different list formats (decimal, alphabetic, Roman, bullets, combinations), including multi-level lists up to 4 levels and a decimal list starting at 5. Each demo uses its own numId and abstractNumId so numbering is independent. For each sample item, the text explicitly states what the numbering should be. For multi-level lists, you also get numbering.xml snippets for all levels shown.

Section 1: Single-level Lists

Decimal list (single-level)

Simple decimal list: 1., 2., 3.

Sample List:

1.Item that should be numbered 1.

2.Item that should be numbered 2.

3.Item that should be numbered 3.

Paragraph XML for first list item:

<w:p>
 <w:pPr><w:numPr><w:ilvl w:val="0"/><w:numId w:val="1"/></w:numPr></w:pPr>
 <w:r>
 <w:t>Item that should be numbered 1.</w:t>
 </w:r>
</w:p>

Numbering XML snippet (all levels used + num mapping):

<w:abstractNum w:abstractNumId="1">
 <w:lvl w:ilvl="0">
 <w:start w:val="1"/>
 <w:numFmt w:val="decimal"/>
 <w:lvlText w:val="%1."/>
 </w:lvl>
</w:abstractNum>
<w:num w:numId="1">
 <w:abstractNumId w:val="1"/>
</w:num>

CSS mapping: CSS: list-style-type: decimal;
